Version 1.3 (2018-01-08)

- Support for scene GameObject references (drag'n'drop)
- Possibility to move scene view into GameObject reference (instead of selecting it)
- Support for creating custom asset lists, with assets of various types, added with drag'n'drop
- Possibility to merge different columns of assets into just one displayed when you use the same column name 
 (for example for many ScriptableObject managers with just one instance)

Version 1.2 (2017-03-10)

- Added saving question popup on scene change. [PR #2]
- Fixed compatibility with newer Unity versions. [PR #1]

Version 1.1.1 (2016-07-09)

- Fix displaying of deleted assets on the Shortcuter window.
- Fix displaying of Shortcuter types on the add item drop down.

Version 1.1 (2016-03-15)

- Moved main project folder to Assets root.
- Added offline documentation.

Version 1.0 (2016-03-01)

- Initial release.
